# ⚠️ ERİŞİM KISITLI / RESTRICTED ACCESS
**Sınıflandırma:** TLP:RED (Strictly Confidential)
**Birim:** Siber Güvenlik Başkanlığı - Operasyonel Teknoloji Güvenliği

Bu dizin altındaki veriler, 256-bit AES katmanı ile şifrelenmiş ham trafik verilerini (PCAP) ve zararlı yazılım imza örneklerini içermektedir. 

### 🛡️ Güvenlik Protokolü
- Verilere erişim sadece **SGB-VPN** üzerinden ve fiziksel **YubiKey** doğrulaması ile mümkündür.
- Yetkisiz dekriptaj (şifre çözme) denemeleri, otomatik olarak sistem alarmını tetikler ve kaynak IP adresi **USOM Kara Listesi**'ne raporlanır.
